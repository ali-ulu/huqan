# AXIOM v0.7.0-rc.1 Release Notes

Branch: `v0.7-causal-wip`

## Scope

AXIOM v0.7 is the causal reasoning line. It focuses on deterministic symbolic causal reasoning instead of probabilistic prediction or UI expansion.

Included:

- causal relations: `CAUSES`, `PREVENTS`, `ENABLES`, `DEPENDS_ON`, `LEADS_TO`
- deterministic causal traversal
- what-if simulator
- causal finalizer summary output
- deterministic demo scenario for `autoLearn default true`

## Known Limitations

- Not a full world model
- No probabilistic prediction layer
- No UI integration on the causal branch
- No enterprise governance layer
- No multi-user permissions or audit framework
- Causal relations are structured inputs, not autonomous discovery
